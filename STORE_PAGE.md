# Bevy Open Siege Store Page

## Short Description

Defend the last greenhouse with tactical plant placement, sun economy, and escalating undead waves.

## Long Description

Bevy Open Siege is a lane-defense strategy game about building a living defensive line under pressure.
Choose from ten specialized plants, manage sun income, adapt to unusual enemy types, and survive a campaign of escalating encounters.

Each level changes the pacing and resource pressure. Early waves teach the core economy and blockers, while later levels introduce faster enemies, armored attackers, healers, jumpers, diggers, frost units, and giant siege threats.

## Key Features

- Ten plant types with distinct tactical roles.
- Ten enemy types with escalating special behaviors.
- Data-driven campaign progression with unlocks and best-score tracking.
- English and Chinese localization.
- Keyboard and mouse controls.
- Pause, fullscreen, volume, and persistent settings.

## Current Release Status

The Linux x86_64 release-candidate package includes the playable 10-level campaign, Blender-generated 3D unit models, production PNG art, synthesized music and sound effects, bilingual localization, store branding assets, and archive smoke tests. Remaining release approval should focus on manual gameplay balance, audio-device QA, and any additional platform packages.

Final store submission also requires the screenshot set defined in `STORE_SCREENSHOTS.md`, with release-package captures covering the title menu, early defense, special enemies, late siege pressure, and victory flow. Use `store_screenshot_check.sh` to capture or validate the screenshot directory before approval.

## Content Disclosures

Bevy Open Siege contains fantasy undead combat with stylized non-realistic visuals. There is no blood, no gore, no gambling, no in-app purchases, no ads, no online multiplayer, no chat, no user-generated content, and no telemetry.
