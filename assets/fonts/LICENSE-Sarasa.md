# SarasaMonoSC-subset.ttf

Subset of **Sarasa Mono SC** (Sarasa Gothic v1.0.39), created with
`pyftsubset`, containing only the glyphs used by the game UI
(printable ASCII plus the characters appearing in the bundled
localization and level data).
