The current playable build uses generated Bevy primitive meshes, project-authored RON data files, fallback SVG branding, AI-assisted PNG art generated specifically for Bevy Open Siege, and project-authored synthesized WAV audio. See `ART_ASSETS.md` for the asset list and generation notes.
No third-party art, music, sound effects, or trademarked game assets are included in the repository; the only third-party asset is the UI font below.
